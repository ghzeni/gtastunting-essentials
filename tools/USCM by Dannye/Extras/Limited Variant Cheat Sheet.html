Overview of Controls/Changes specifically made to the "Legacy/Limited variant" in v4c

This variant has always supported only a small subset of the complete USCM feature set, however with this release the functionality of this variant has finally been fleshed out a bit.

Note: The name of this variant has been renamed (again) slightly to just "Limited" for this release.

Feature Controls

With all the newly added/supported features, for their button combinations I've tried to get the right mix of being easy enough to trigger intentionally (with elements of consistency with other/existing features to assist here), while also being out-of-the-way enough to hopefully be able to ignore/forget-about them when unneeded and never worry about accidentally triggering them during normal stunting behaviour or when trying to interact with other scm features. That many of the additions also belong to features that likely don't see frequent (or at least certainly not urgent-on-the-fly) usage, will hopefully also help in these regards.

Default Feature States

Only one of these were ever different in Legacy/Limited compared to all the other variants. In this case the corresponding feature which allows the state to be changed at-will is now supported in Legacy/Limited.
As such, the changes noted in the mockup below make it so all feature defaults are the same across all variants.
Note that for any toggles, a save-game will remember the state they are in, so an undesirable default doesn't have to get in the way on each stunting session, as a save can be prepared/loaded with everything in a preferred state.

Feature Default State Notes on Changes/Additions
Player vulnerability Invulnerable
Vehicle vulnerability Invulnerable

Wanted Level 0 Stars (Player Ignored by NPCs) (used to be 'Player Not Ignored', which seems irrelevant now that you can manually opt-in to that behaviour with the via the Wanted Level feature)
Prior vehicle preference Last vehicle

Stats Overlay Off

Automatic Features

  • Unlimited Player Health/Armour (includes not being decreased by Fire)
  • Unlimited Player Sprint
  • Player cannot be carjacked (this used to not be true of Legacy/Limited variant, I have no idea why, so now its consistent with everyone else)
  • Perfect Weapon Accuracy
  • Taxi Boost & Nitro available (press Horn while driving for Boost/Jump)
  • "Unique Jump" slow-motion/camera removed
  • Various object pickups (rampages, hidden packages, etc) removed
